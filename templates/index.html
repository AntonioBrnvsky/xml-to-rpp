XML to RPP converter

This is a simple tool to convert FCPX-type xml files to Reaper's rpp sessions. It can be used to exchange session data between editors and sound mixers.

To convert a XML file, simpy choose it via dropdown menu and press Convert button below.

After file has been converted, you will have to point Reaper to the media files.

Please note that if audio from video used in sequence, you will have to point Reaper to that video file. To counter that, I advise to bounce audio from video files used in sequence prior to exporting XML from your video editing software.
